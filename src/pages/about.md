**AI Maker Lab** は、AIで個人が「実際に作って公開する」ためのチュートリアル＋比較サイトです。

## サイトの方針

このサイトは、AIで個人がプロダクト・コンテンツを作り、副業や本業として稼ぐためのツール群を、**個人開発者の実体験ベース**で紹介します。

企業向けの汎用比較ではなく、「実際に触ってみて、何ができて何ができないか」を正直に書くことを心がけています。

## 3層ペルソナ

このサイトは3つの読者層を想定しています。

### 👨‍💻 層1：エンジニア（AIで開発を加速したい）
Cursor / Claude Code / Cline / Windsurf / v0 / MCP / Agent SDK / n8n の深掘り。

### 🛠️ 層2：AI開発挑戦中（非エンジニアも含む）
Lovable / Bolt / Replit Agent などの Vibe Coding ツール入門。「初めての AI でアプリ開発」ガイド。

### 💰 層3：AIで稼ぎたい個人
副業ロードマップ（0→月1万円→月10万円）、収益化事例集、AI動画・音声・コンテンツ販売の選び方。

## AI診断

サイトには **AI診断** 機能があります。4-5問の質問に答えると、あなたに合うAIツール3つを提示します。

## 運営者について

個人開発者として、ここで紹介するツールを実際に使い込んだ経験を発信しています。
