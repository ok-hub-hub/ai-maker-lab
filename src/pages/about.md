**AI Maker Lab** は、Vibe Coding（v0 / Lovable / Bolt / Cursor）と AI動画・AI音声・業務自動化のツールを、**個人開発者が実際に触ったうえで公開記録として書きとめる** ジャーナルです。

## このサイトの差別化軸

世の中には AI ツール紹介サイトが数えきれないほどあります。その中で、AI Maker Lab が他と違うのは次の3点です。

### Ⅰ ─ 実体験ベース
触っていないツールは扱いません。料金プラン、無料枠の限界、有料に上げるべきタイミング、日本語耐性まで、**自分で課金して試した結果**を書きます。

### Ⅱ ─ 公開記録
失敗も含めて公開します。「Vibe Coding で SaaS を3日で作って公開してみたら何が起きたか」のような、結果が予測不能な実験記録を、加工せずに書きとめます。

### Ⅲ ─ 個人開発者の現実視点
企業向けの汎用比較ではなく、「**ひとりで作って、ひとりで公開して、ひとりで収益化する**」 ことを前提にした評価軸で書きます。チーム前提の SaaS や、PR 記事の言い回しは使いません。

## サイトの方針

このサイトは、AIで個人がプロダクト・コンテンツを作り、副業や本業として稼ぐためのツール群を、**個人開発者の実体験ベース**で紹介します。

企業向けの汎用比較ではなく、「実際に触ってみて、何ができて何ができないか」を正直に書くことを心がけています。

## 3層ペルソナ

このサイトは3つの読者層を想定しています。

### Ⅰ ─ For Engineers
Cursor / Claude Code / Cline / Windsurf / v0 / MCP / Agent SDK / n8n の深掘り。AIで開発を加速したい層へ。

### Ⅱ ─ For Builders
Lovable / Bolt / Replit Agent などの Vibe Coding ツール入門。非エンジニアも含む「はじめての AI アプリ開発」を支える章。

### Ⅲ ─ For Earners
副業ロードマップ（0 → 月1万円 → 月10万円）、収益化事例集、AI動画・音声・コンテンツ販売の選び方。

## AI診断

サイトには **AI診断** 機能があります。4-5問の質問に答えると、あなたに合うAIツール3つを提示します。
