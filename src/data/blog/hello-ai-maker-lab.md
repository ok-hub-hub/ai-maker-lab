AI Maker Lab は、AIで個人が「実際に作って公開する」ことに特化した実体験ジャーナルです。**Claude Code で「AI 会社」を1ヶ月運営した個人開発者** が、**AI 駆動の編集体制** で記録します。サイト構築は **Astro 本体は手書きベース／LPプロトタイプは v0 で試作、記事生成・運用は Claude Code 駆動** で進めています。

## このサイトでやらないこと

- 企業向けの汎用AIツール紹介
- AIニュースの単純な転載
- 触ったことのないツールのアフィリエイト誘導

## このサイトでやること

- 個人開発者向けの **公開情報ベースの解説チュートリアル**（「Lovable で SaaS をつくる公式手順をまとめてみた」）
- ツール群の **3層ペルソナ別ガイド**：
  - **Ⅰ ─ For Engineers** — Cursor / Claude Code / v0 / MCP / n8n
  - **Ⅱ ─ For Builders** — Lovable / Bolt / Replit Agent（非エンジニア向け Vibe Coding）
  - **Ⅲ ─ For Earners** — 副業ロードマップ、収益化事例
- **AI診断** — 4-5問の質問に答えると、あなたに合うAIツール3つを提示
