## なぜ「Day1」なのか：構築フェーズの終わり

[Claude Code で AI 会社を運営する](/posts/claude-code-ai-company-1month-2026/) 1ヶ月で、私たちが作ったのは「**動くメディア**」でした。

- 3サイト（kokolabo / careerlounge / ai-maker-lab）の構築・公開
- 法務6ページ（pr-disclosure / disclaimer / privacy / terms / about / contact）の完備
- /go/<id>/ リダイレクトパターンの導入（3サイト合計 88 CTA リンク）
- CTR 改善の入った10記事 title / description 反映
- Pinterest / Threads / X / Instagram の SNS 4チャネル開設

これら全部は「**マネタイズの土台**」です。土台があっても、**収益化の蛇口を開かない限り、月収益は ¥0 のまま**。これが「構築フェーズの終わり」と「マネタイズ Day1」の境界線でした。

## AI 会社が辿った「ゼロから1への移行」5ステップ

具体的に何を AI 会社（Claude Code）にやらせたか。

### ステップ1：Amazon アソシエイトの完全網羅（最初の蛇口）

最初の収益化ソースは Amazon アソシエイト（`tag=aimakerlab26-22`）です。これは審査が比較的緩く、ASP に頼らず始められる「**ゼロから 1**」の最有力候補です。

AI 会社の Claude Code に「kokolabo / careerlounge / ai-maker-lab の全記事を grep して、Amazon リンク網羅率 100% を達成して」と指示しました。結果、**3サイト合計 236箇所**で `aimakerlab26-22` タグ付き Amazon リンクが稼働している状態を担保しました。

ただし Amazon の紹介料率は **1〜3%**（書籍3%、健康食品2%、家電2-4%）と低めです。**月10万円達成には Amazon 単独では非現実的**で、客単価 ¥3,000 × CVR 1% × 月50万PV が必要になります。これは流入量で言うと、現状の50-100倍。

### ステップ2：A8.net 既存稼働の発掘

次に、Claude Code に **A8.net の既ログイン状態**を確認させました。Playwright を経由して `https://media-console.a8.net/home` を navigate すると、**オーナーのアカウントが既ログイン**状態で保持されていることが判明。

既稼働中の A8 アフィリリンクは5本：
- ConoHa AI Canvas
- Xserver / Xserver ビジネス（2件）
- mixhost
- お名前.com

すべて `a8mat=4B3OR0` で始まる、AI Maker Lab オーナーの A8 媒体 ID で識別されるリンクです。これらは ai-maker-lab のホスティング・ドメイン関連記事から既に流入を受け取る形で稼働中。

### ステップ3：A8 提携申請を「自動化で大量実行」

「既稼働中 A8 リンクは 5本だけ」という観察から、Claude Code に **大量の追加申請を代行**させました。

カテゴリ別の申請件数（5/24 深夜〜5/25 深夜の累計）：

| カテゴリ | 件数 | 主要プログラム |
|---|---|---|
| 医療脱毛系 | 3件 | フレイアMEN / シロノクリニック / ルシアクリニック |
| 転職エージェント系 | 10件 | 主要転職サイト群 |
| プラセンタ系 | 5件 | 健康食品メーカー群 |
| ローヤルゼリー系 | 4件 | 養蜂・健康食品メーカー |
| コラーゲン系 | 3件 | コラーゲン専門ブランド |
| **合計** | **25件** | — |

申請の自動化フローは以下の通り：

```
1. キーワード検索（"医療脱毛"等で A8 内検索）
2. 未提携プログラム ID を一括取得
3. プログラム詳細ページに個別 navigate
4. 「プログラムの提携申請をする」ボタンを click
5. 「提携申請完了」ページに遷移、ステータス確認
6. 次のプログラム ID へ
```

1件あたり 約 15-20 秒。25件で **約 8分** の連続作業を Playwright が自動でこなしました。

### ステップ4：即時提携プログラムの即実装

申請した 25件のうち、1件だけが「**即時提携**」で承認されました。**シロノクリニック（定額制全身医療脱毛）** です。報酬は新規来院 ¥4,545 ／新規施術 ¥13,986。

これを実際に「ゼロから 1」を作る最初のリンクとして実装します：

1. A8 管理画面で「広告リンク作成」→ アフィリ URL を取得
2. kokolabo の `src/data/vendors.ts` に shirono エントリを追加（id / name / officialUrl / affiliateUrl の4フィールド）
3. /go/[id].astro 動的ルートが自動で `/go/shirono/` を生成、affiliateUrl が未設定なら officialUrl にフォールバックする設計
4. 既存の `medical-vs-salon-hair-removal.astro` 記事の末尾に「定額制プランも検討」セクションを追加、`/go/shirono/` リンクを2箇所埋め込む（テキストリンク＋CTAボタン）
5. ビルド → Cloudflare Pages デプロイ → 本番反映確認

ここまでで **「kokolabo 医療脱毛比較記事から A8 経由で収益発生する」最初のリンクが稼働開始**しました。

### ステップ5：シロノリンクを医療脱毛系全記事に展開

「1記事だけだとクリック発生確率が低い」ことに気付いて、AI 会社に **同じシロノ広告リンクを医療脱毛系の他3記事にも展開**させました：

- `vio-medical-hair-removal-ranking`（全身脱毛＋VIO 検討者向け）
- `student-discount-medical-hair-removal-ranking`（学割では補えない長期予算）
- `summer-rush-medical-hair-removal-ranking`（来年・再来年の夏に備える視点）

各記事に補足セクションを追加し、合計 **7箇所**でシロノクリニックの広告リンクが稼働する状態へ。これでクリック発生確率がほぼ4倍に拡大。

## 即時提携 vs 審査制：A8 で最初に動かしたこと

A8.net のプログラムには「**即時提携**」と「**審査制**」の2種類があります。

| 種類 | 承認時間 | 戦略的意味 |
|---|---|---|
| 即時提携 | クリック直後 | 「ゼロから 1」の最速ルート。今夜稼働開始可能 |
| 審査制 | 1日〜数週間 | メディアサイトの記事数・コンテンツ品質を見た上で承認・否認 |

今夜の戦略は **「即時提携を即稼働 + 審査制を一括申請」** でした。シロノクリニックが偶然「即時提携」だったおかげで、Day1 で稼働するリンクが生まれました。残り24件は審査中のまま。5/26-5/27 で多くが承認される見込みです。

## マネタイズの方程式：なぜ Day1 ＝ 月0円なのか

マネタイズの基本式は以下：

```
月収益 = 月間PV × CTR × CVR × 平均報酬
```

**現状（Day1 終了時点）の数字**：
- 月間PV：約 200（過去24日で 20-30セッション。Microsoft Clarity 実測）
- CTR：5%（シロノ補足セクションが見られた場合のクリック率推定）
- CVR：1%（医療脱毛は高単価で意思決定までの離脱が大きい）
- 平均報酬：シロノ ¥4,545 / Amazon ¥30-200

最大値の楽観試算でも：
**月収益 = 200 × 5% × 1% × ¥4,545 = ¥454**

つまり Day1 終了時点では **月収益は良くて ¥500 程度**です。

これが現実で、「**1ヶ月で月10万円なんて無理**」と AI 会社は正直に出してくれます。

## 失敗の予感：審査が下りても収益がゼロのままになるシナリオ

仮に 5/26-5/27 で A8 24件すべてが承認されても、**月10万円達成は遠い**です。理由は3つ：

### 1. 流入の絶対量不足

Microsoft Clarity の実測で、3サイト合計 **過去7日で56セッション**。Threads から ai-maker-lab に7セッション/週流入があったのが唯一の SNS チャネル実証でしたが、これも月30セッション程度。検索流入はほぼゼロ（Google Search Console のクリック数 = 0）。

**月収益¥10万 = 月間PV 数十万 × CTR ≧ 5% × CVR ≧ 1% × 平均報酬 ≧ ¥3,000** という方程式で逆算すると、**月50万PVが必要**。現状の 2500倍。

### 2. SEO 育成期間（検索流入は最低 3-6ヶ月）

検索流入が増えるには：
- サイトマップが Google にクロールされる（1-7日）
- ページがインデックスされる（数日〜数週間）
- 検索順位が上位に上がる（最低 3-6ヶ月、業界的には 6-12ヶ月）

つまり **「今日記事を書いて、明日検索流入が爆増する」ことはない**。Day1 の流入の絶対量不足はしばらく続きます。

### 3. CVR の低さ

医療脱毛のような高単価系は、SERP からクリック → 詳細記事 → CTA クリック → 公式サイト遷移 → 無料カウンセリング予約 → 来院、という長いファネルです。**CVR は1%以下が現実**です。

転職エージェント系は **CVR 2-3%** の可能性があるが、これも記事数と SEO 育成が前提。

## 6-12ヶ月で月10万円が現実射程と言える理由

ただし、ここで諦める理由はありません。**6-12ヶ月後** には現実射程に入る根拠：

### 数字での裏付け

| 期間 | 月間PV（推定） | CTR | CVR | 月収益 |
|---|---|---|---|---|
| Day1（5/25） | 200 | 5% | 0% (蛇口開ばかり) | ¥0 |
| 1ヶ月後 | 1,000 | 5% | 0.5% | ¥500-¥1,000 |
| 3ヶ月後 | 5,000 | 5% | 1% | ¥10,000-¥30,000 |
| 6ヶ月後 | 30,000 | 5% | 1.5% | ¥40,000-¥80,000 |
| 12ヶ月後 | 100,000 | 5% | 2% | **¥100,000-¥200,000** |

これは「**毎月10本の記事追加 + 検索順位向上 + SNS 流入育成 + ASP 最適化**」を継続できた場合のシナリオです。

### 必要なコミットメント

- **記事の継続執筆**（月10本ペース、AI 会社で半自動化）
- **SNS チャネル育成**（Threads / X / Pinterest / Instagram / note の継続発信）
- **CTR / CVR 改善**（GSC データを毎月見て title / description 改修）
- **記事内導線最適化**（CTA 位置・コピー・色のテスト）

これを「個人開発者ひとり＋ AI 会社」でやり切れるか？という挑戦の Day1 が今日です。

## 実装の現場：どの記事のどこにリンクを置いたか

参考までに、本記事と並んで Day1 のマネタイズ第1号となった **kokolabo の医療脱毛比較記事**にどんな実装をしたか公開します（実は本サイト ai-maker-lab とは別の姉妹サイトです）：

### kokolabo `medical-vs-salon-hair-removal.astro` の補足セクション

記事末尾に **「定額制プランも検討する場合」** という補足セクションを追加。本文の比較対象（リゼ・エミナル・フレイア・TCB・湘南の5院）とは別軸の「定額制」というポジショニングで、シロノクリニックを差別化して紹介。

CTA の文言：
- 「シロノクリニックの定額制プランを見る →」（CTAボタン）
- 「シロノクリニック公式」（テキストリンク・本文中）

PR 表記：「※ PR：本リンクは A8.net 経由のアフィリエイトリンクです」を末尾に明示。景表法ステマ規制（2023年10月施行）準拠。

## 次の章へ：Day2 以降の打ち手

Day1 で「**蛇口を開ける**」は完了しました。Day2 以降の打ち手は：

### Day2-7：審査結果が出揃う

A8 24件のうち承認されるものが順次出てきます。承認分を `vendors.ts` / `agents.ts` / `tools.ts` の `affiliateUrl:` フィールドに1行追加するだけで、3サイトの 88 CTA が順次マネタイズ稼働します。

月10本ペースで記事追加。CTR 改善した10記事の SERP 反映を待ちつつ、新規記事も同じ方針で執筆。GSC を週1で観察。

### Month 6-12：本格収益化

検索流入が累積し、PV 月10万を超えるあたりから、**月10万円達成が現実射程**に入ります。
