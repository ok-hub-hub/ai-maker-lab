## キャリアカンパニーとは（運営・サービス内容）

キャリアカンパニーは、**未経験からITエンジニアを目指す人向けの転職エージェント**（運営：株式会社エンデバース）です。キャリア相談・求人紹介・選考対策を受けられます。

転職エージェントの収益モデルは「採用企業が紹介料を支払う」形で、**求職者側は無料で利用できます**。

未経験・スキル0からの相談に対応していると明記されており、オンラインで無料転職相談を予約できます（来社必須ではない）。詳細は[公式サイト](/go/career-company)で最新情報をご確認ください（本記事記載の情報は 2026-06-29 取得）。

## 良い評判・転職エージェントとしての強み

1. **求職者は完全無料**：転職エージェントのビジネスモデルは採用企業からの報酬です。求職者が費用を払うことはありません。
2. **未経験・スキル0からの相談に対応**：「何も知らない状態で相談していいの？」という不安は不要。現状を正直に話す場として使えます。
3. **オンラインで相談できる**：来社不要でオンライン予約できるため、在職中・地方在住でも動きやすい。
4. **選考対策まで支援**：求人紹介にとどまらず、書類添削・面接対策まで伴走する体制が特徴として挙げられています。

## 正直な弱点・注意点

1. **求人はエリア・経歴により異なる**：紹介制のため、希望するポジションや地域によって紹介できる案件が限られる場合があります。特にリモート希望の場合は事前に確認を。
2. **インフラ・クラウド系求人が中心になりやすい傾向**：未経験向けIT求人はインフラ・サポート系が多い傾向があります。開発職志望の場合は、紹介案件の傾向を相談段階で確認してください。
3. **転職や年収を保証するものではありません**：転職エージェントは機会と支援を提供するサービスであり、転職成功や収入向上を約束するものではありません。結果は本人の状況・市況・取り組みによります。
4. **相談は無料だが、複数エージェント並行が定石**：1社に絞らず、複数の窓口で情報を集めるのが転職活動の基本です。

## 料金（求職者は無料）

| 項目 | 費用 |
|------|------|
| 相談・カウンセリング | **無料** |
| 求人紹介・マッチング | **無料** |
| 書類添削・面接対策 | **無料** |
| 求職者の自己負担 | **なし** |

費用は採用企業側が負担するモデルのため、求職者は一切無料です。スクールとは異なり、「相談だけ受けてみる」ことに経済的リスクはありません。

## 向く人 / 向かない人

| | キャリアカンパニー |
|--|--|
| **向く人** | 「今すぐIT業界に入れる可能性を知りたい」未経験者。エージェントに伴走してもらいながら転職活動したい人。費用をかけずに転職サポートを受けたい人 |
| **向かない人** | まずじっくりスキルを積んでから転職したい人（→ 先にスクールを検討）。希望職種・エリアが非常に限定的な人（紹介案件が合わない可能性あり） |

## 転職エージェントとスクール、どっちを先に使う？

| | **転職エージェント**（キャリアカンパニー） | **スクール**（ポテパンキャンプ／忍者CODE） |
|--|--|--|
| **費用（求職者）** | 無料 | 相談は無料、受講は有料 |
| **向く目的** | 「今の自分で転職できるか知りたい」 | 「スキルをつけてから転職したい」 |
| **時間軸** | 比較的早期に転職活動を開始 | 数ヶ月の学習期間後に転職活動 |
| **最初の一歩** | 無料転職相談 | 無料カウンセリング |

**迷ったら、まず転職エージェントに無料相談**するのが効率的です。「今の経歴・スキル感で転職できる見込みがあるか」を知ってから、スクールに行くかどうかを決めれば、無駄がありません。

## FAQ

**Q. 無料相談だけ受けて、申し込まなくてもいい？**
A. 大丈夫です。相談・カウンセリングは情報収集の場で、契約ではありません。合わなければ進めなくてOKです。

**Q. 本当にIT転職できる？ 未経験でも？**
A. 転職を保証することはできません（どのサービスも同様です）。「相談に対応している」というのは入口であり、実際に転職できるかは経歴・エリア・市況・取り組みによります。まず無料相談で**自分のケース**を確認してください。

**Q. スクールに行かなくても転職できる？**
A. 「未経験でそのまま転職できる求人があるかどうか」は状況次第で、無料相談で確認するのが確実です。スキルが不足していると判断された場合に、スクール利用を勧めることもあります。

**Q. 転職エージェントとスクールを同時に使っていい？**
A. 問題ありません。転職エージェントで「どんなスキルが求められているか」を把握してからスクールを選ぶ、という順番も有効です。

**Q. 年齢制限はある？**
A. 公式サイトで最新の対象条件をご確認ください（本記事では断定しません）。未経験向けIT求人は一般的に20代が有利な傾向がありますが、詳細は相談段階で確認するのが確実です。
