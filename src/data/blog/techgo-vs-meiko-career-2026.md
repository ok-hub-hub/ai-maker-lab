「**AIを使えるようになった。この力、転職市場でどれくらい評価されるんだろう？**」

Claude Code や Cursor を日常的に回すようになると、一度はよぎる問いだと思います。私たちも [AI会社を1ヶ月運営した実録](/posts/claude-code-ai-company-1month-2026/) を書いたあと、同じ問いに突き当たりました。

答えを自分で出すのは無理です。**市場価値は、求人を実際に持っている人に聞くしかない**。そこで候補になるのが、どちらもITエンジニア転職に特化した **TechGo（テックゴー）** と **明光キャリアパートナーズ** です。本記事では、この2つの分かれ道を整理します。

> **広告に関する表記**：本記事には [TechGo](/go/techgo)・[明光キャリアパートナーズ](/go/meiko-career) を経由するリンクが含まれます。リンク経由での面談申込により、AI Maker Lab に紹介料が発生する場合があります。**面談・相談はどちらも求職者は無料**で、比較の中身は公平な視点で書いています。

## 30秒で結論：分かれ道は「実務経験の有無」1つだけ

| あなたの状態 | 相談先 |
| --- | --- |
| **エンジニアとしての実務経験がある**（業務でコードを書いている・書いていた） | **[TechGo](/go/techgo)** — 経験者のハイクラス・上流転職に特化 |
| **実務は未経験**（独学・個人開発・AIツール経験のみ） | **[明光キャリアパートナーズ](/go/meiko-career)** — 未経験〜ジュニアのIT転職に特化 |
| どちらか微妙（業務での開発経験が浅い・ブランクがある） | 先に [明光の無料面談](/go/meiko-career) で現在地を聞き、経験者枠で戦えると言われたら [TechGo](/go/techgo) にも相談 |

料金はどちらも**求職者は無料**です（転職エージェントは採用企業側から報酬を得るモデル）。だからこの比較は「どちらにお金を払うか」ではなく、「**どちらの土俵なら自分の話がかみ合うか**」の比較になります。

## TechGo（テックゴー）とは

**ITエンジニア経験者のハイクラス転職に特化した転職エージェント**です。エンジニアのキャリアを熟知したアドバイザーが、求人提案と企業別の選考対策で年収アップ・キャリアアップを支援する、と公式が掲げています。

- **向く人**：実務経験があり、上流工程・年収アップ・より良い環境を狙う人
- **AIスキルとの相性**：2026年の経験者市場では「AIツールを業務でどう使い、何を改善したか」を語れる人がまだ少数派です。生成AIで開発フローを変えた経験は、経験者転職の面談で具体的な武器になります
- **注意点**：経験者向けのため、実務未経験だと紹介できる求人が限られる可能性が高い

## 明光キャリアパートナーズとは

**未経験〜ジュニア層のエンジニア転職に特化した転職エージェント**です。「AIツールに触れてITで働くことに興味が出た」段階の人が、キャリア面談から次の一歩を検討できます。

- **向く人**：実務未経験でIT/エンジニア職を目指す人、独学や個人開発の経験をどう職務経歴に翻訳すればいいか分からない人
- **AIスキルとの相性**：未経験市場では「学習意欲と継続の証拠」が最重要です。AIツールで作ったもの・続けてきた記録は、面談でそのまま材料になります（[実録：AIスキルは職務経歴になる](/posts/claude-code-ai-company-1month-2026/)）
- **注意点**：紹介制のため、経歴・エリアによって紹介できる求人が限られる場合があります

## 面談前に用意しておくと話が早いもの

どちらに相談する場合も、**「AIで何をしたかの記録」**を持っていくと面談の質が変わります。

1. **作ったもの・自動化したもの**のリスト（個人開発でも業務改善でも。URLやスクリーンショットがあれば強い）
2. **使えるツールと使い方のレベル感**（「ChatGPTに質問できる」と「Claude Codeで開発フローを回せる」は別物として扱われます）
3. **どれくらいの期間続けているか**（継続の証拠は未経験転職ほど効きます）

この「AIスキルを職務経歴に翻訳する」考え方は、[未経験からITエンジニア転職 完全ガイド](/posts/it-engineer-career-start-2026/) でも詳しく書いています。

## FAQ

**Q. 転職するか決めていないのに面談を受けていい？**
A. 大丈夫です。面談は「転職の意思表明」ではなく情報収集の場で、合わなければ進めなくてOKです。現在地を知ると「今の環境に残る」判断にも根拠が持てます。

**Q. 両方に相談してもいい？**
A. 問題ありません。ただし自分の経験レベルに合わない側は話がかみ合いにくいので、まず上の表で土俵を選ぶのが時間の節約になります。

**Q. 無料なのはなぜ？**
A. 転職エージェントは採用が決まった際に**企業側から**報酬を受け取るモデルのためです。求職者側の費用負担はありません。

**Q. 転職や年収アップは保証される？**
A. されません。どちらも相談・紹介サービスであり、結果は本人の経歴・市況・取り組みで変わります。
