「ブログを始めたい。でもサーバー？ドメイン？WordPress？……調べるだけで日が暮れる」

この記事は、その「調べる時間」をAIに肩代わりさせる開設ガイドです。**各ステップに、ChatGPT や Claude にそのまま貼れるプロンプトを同梱**しました。分からない単語が出たら、調べずにAIに聞く——これが2026年のブログ開設の最短ルートです。

> **広告に関する表記**：本記事には [/go/xserver](/go/xserver) および [/go/conoha](/go/conoha) を経由するアフィリエイトリンクが含まれます。リンク経由でご契約いただくと、当サイトに紹介料が発生する場合があります。手順と中身は独立した立場で執筆しています。

## 全体像：やることは5つだけ

| Step | やること | 所要 |
|---|---|---|
| 1 | レンタルサーバーを契約する | 10分 |
| 2 | 独自ドメインを決める（無料特典でOK） | 5分 |
| 3 | WordPressをセットアップする | 5分 |
| 4 | 最初の記事をAIと書く | 10分 |
| 5 | 計測（GSC/GA4）を設置する | 5分＋後日 |

## Step 1：レンタルサーバーを契約する（10分）

WordPressブログの土台です。迷うのは2択だけ：

- **[エックスサーバー](/go/xserver)**：国内シェアNo.1（公式サイト表記）・2003年から運用。執筆時点で30%OFFキャンペーン中（公式記載 2026-07-06 まで）
- **[ConoHa WING](/go/conoha)**：36ヶ月契約で月額約660円〜の最安級・無料ドメイン2個

どちらでも失敗しません。決め方は「何年やるか」だけ。詳しい比較は [エックスサーバー vs ConoHa WING 徹底比較](/posts/xserver-vs-conoha-wing-2026/) にまとめました。

**🤖 迷ったらAIに貼るプロンプト：**

```
あなたはブログ開設のアドバイザーです。私の状況に合うレンタルサーバーを選んでください。

- 目的：[YOUR_PURPOSE 例：副業でアフィリエイトブログ]
- 続ける自信：[YOUR_CONFIDENCE 例：1年は必ずやる／まず3ヶ月試したい]
- 月予算：[YOUR_BUDGET 例：1,000円前後]
- 候補：エックスサーバー（スタンダード）と ConoHa WING（ベーシック）

それぞれの契約期間別料金の考え方と、私の場合どちらの何ヶ月契約が合理的か、理由つきで結論を出してください。
```

> 💡 プラン選びの注意：最安の36ヶ月一括は「3年続ける前提」の価格です。自信がなければ12ヶ月が無難（[比較記事の「やめるとき」の話](/posts/xserver-vs-conoha-wing-2026/) 参照）。

## Step 2：独自ドメインを決める（5分）

サーバー契約時に**無料ドメイン特典**（エックスサーバー1個／ConoHa WING2個）が付くので、別途買う必要はありません。

**🤖 ドメイン名を10案出させるプロンプト：**

```
ブログの独自ドメイン名の候補を10個提案してください。

- ブログのテーマ：[YOUR_THEME 例：AIツールの活用と副業記録]
- 希望の雰囲気：[YOUR_TONE 例：信頼感がある／覚えやすい／日本語ローマ字OK]
- 条件：.com を第一候補、8〜15文字、ハイフンなし、商標と紛らわしくない

各案に「読みやすさ」「覚えやすさ」「検索時の見え方」の短評をつけ、最終的に上位3つを推薦してください。
```

決めたら、サーバーの申込画面でそのまま取得します。深く悩まないこと——**ドメイン名で順位は決まりません**（詳しくは [ドメイン選びガイド](/posts/domain-selection-guide-2026/)）。

## Step 3：WordPressをセットアップする（5分）

両サーバーとも「WordPressかんたんセットアップ／クイックスタート」があり、**申込フォームの中でWordPress導入まで完了**します。SSL（https化）も自動です。

**🤖 初期設定をチェックリスト化するプロンプト：**

```
WordPressを開設直後にやるべき初期設定を、優先度順のチェックリストにしてください。

- サーバー：[YOUR_SERVER 例：エックスサーバー スタンダード]
- テーマ：[YOUR_THEME 例：Cocoon（無料）]
- 目的：[YOUR_PURPOSE 例：アフィリエイトブログ]

各項目は「設定場所（管理画面のどこか）／推奨値／なぜ必要か1行」の形式で。
パーマリンク設定・SSL確認・不要プラグイン削除・コメント設定・ニックネーム設定（本名露出防止）を必ず含めてください。
```

> ⚠️ 1つだけ手動で必ず：**ユーザー名がそのまま表示される設定になっていないか**（ニックネーム設定）。本名・ログインIDの露出はセキュリティと匿名性の両方で事故になります。

## Step 4：最初の記事をAIと書く（10分）

完璧な1本目を狙わないこと。最初の記事の役割は「サイトが動いていることの確認」と「検索エンジンへの挨拶」です。

**🤖 最初の記事を一緒に作るプロンプト：**

```
ブログの最初の記事を一緒に作ってください。

- ブログテーマ：[YOUR_THEME]
- 読者：[YOUR_READER 例：これからAIツールを使ってみたい会社員]
- 私の実体験：[YOUR_EXPERIENCE 例：ChatGPTを業務で半年使っている]

条件：
1. タイトルは疑問形で40文字以内
2. 構成は「読者の悩みに共感→結論→理由→具体例（私の実体験を使う）→まとめ」
3. 1,500〜2,000字、見出し（H2）3〜4本
4. 私の実体験が入る場所を [ここに体験談] とマークして、何を書くべきか指示を添える

まず構成案だけ出して、私がOKしたら本文を書いてください。
```

> 💡 ポイントは条件4。**AIに全部書かせず、体験談の枠を自分で埋める**——これが2026年の検索評価（一次情報重視）への最短対応です。当サイトの[実験データ](https://note.com/aimakerlab/n/nd6119b00b661)でも、AI任せの量産記事は評価されないことを実証済みです。

なお、ChatGPT ではなく**日本語ブログ向けの専用ライティングツール**で叩き台を量産する手もあります。テンプレ選択式で手が止まらない [Catchy（無料10クレジットで試せます）](/go/catchy) が定番で、実際の使用感は[Catchy の詳細レビュー](/posts/catchy-review-2026/)にまとめています。

## Step 5：計測を設置する（5分＋後日）

書いた記事が「誰に・どの検索語で」届いたかを見る装置です。**開設初日に入れておくと、後で必ず効きます**。

**🤖 計測設置を案内させるプロンプト：**

```
WordPressブログに Google Search Console と GA4 を設置する手順を、今日できる最小構成で案内してください。

- サーバー：[YOUR_SERVER]
- テーマ：[YOUR_THEME]
- スキルレベル：HTMLはコピペならできる

「所有権の確認はどの方式が最も簡単か」「GA4の測定IDをどこに貼るか」「設置後、最初に見るべき数字は何か（週1回・5分で）」の3点を必ず含めてください。
```

## 完走チェックリスト

- [ ] ニックネーム設定（本名露出なし）
